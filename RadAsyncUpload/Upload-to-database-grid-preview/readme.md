#RadAsyncUpload Upload images to a database and preview inside RadGrid

###Description
This demo shows how to store image binary data, file name and other additional information inside database. 
Data is displayed inside RadGrid as RadBinaryImage column is used.

###How to run the demo?
1. Copy Telerik.Web.UI.dll into the Bin folder.
2. Configure the Connection String
